Ketika user meminta untuk memahami alur kerja, sistem aplikasi, atau dokumentasi lengkap, ikuti langkah-langkah berikut secara berurutan:

1. Baca referensi dokumen utama yang berada di direktori `/home/alice/Migunani_Project/docs/`. 
   Dokumen yang harus diprioritaskan:
   - `/home/alice/Migunani_Project/docs/MEKANISME_SISTEM.md`
   - `/home/alice/Migunani_Project/docs/SOP_ALUR_SISTEM.md`
   - `/home/alice/Migunani_Project/docs/RBAC_DOCUMENTATION.md`
   - `/home/alice/Migunani_Project/docs/GUIDE_APLIKASI_MIGUNANI_MOTOR.md`

2. Untuk dokumentasi teknis backend (API Contract, Database Schema, dsb.), baca file pada ruang `/home/alice/Migunani_Project/back_end/docs/`.

3. Gunakan tool `view_file` pada file `.md` yang relevan dari direktori-direktori tersebut sebelum memberikan jawaban yang valid mengenai alur kerja aplikasi.

4. Berikan rangkuman atau penjelasan yang tepat berdasarkan panduan dalam dokumen tersebut.
